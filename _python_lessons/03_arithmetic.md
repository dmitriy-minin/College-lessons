## Основные операторы

Python поддерживает стандартные арифметические операции:

| Оператор | Описание | Пример | Результат |
|----------|----------|--------|-----------|
| `+` | Сложение | `5 + 3` | `8` |
| `-` | Вычитание | `10 - 4` | `6` |
| `*` | Умножение | `6 * 7` | `42` |
| `/` | Деление | `15 / 4` | `3.75` |
| `//` | Деление нацело | `15 // 4` | `3` |
| `%` | Остаток от деления | `15 % 4` | `3` |
| `**` | Возведение в степень | `2 ** 8` | `256` |

```python
a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.3333333333333335
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000
```

## Приоритеты операций

Python вычисляет выражения в строгом порядке. Правило такое же, как в математике: сначала степени, потом умножение и деление, затем сложение и вычитание.

| Приоритет | Операция |
|-----------|----------|
| 1 (высший) | `**` (степень) |
| 2 | `*`, `/`, `//`, `%` |
| 3 (низший) | `+`, `-` |

Операции с одинаковым приоритетом выполняются **слева направо**.

```python
result = 2 + 3 * 4
print(result)  # 14, не 20 (сначала 3 * 4 = 12, потом 2 + 12 = 14)

result = 10 - 4 - 2
print(result)  # 4 (слева направо: 10 - 4 = 6, 6 - 2 = 4)

result = 2 ** 3 ** 2
print(result)  # 512 (степень: справа налево: 3**2 = 9, 2**9 = 512)
```

### Скобки меняют порядок

Чтобы изменить порядок вычислений, используйте круглые скобки:

```python
result = (2 + 3) * 4
print(result)  # 20 (сначала 2 + 3 = 5, потом 5 * 4 = 20)

result = (10 - 4) * (2 + 1)
print(result)  # 18

# Вложенные скобки
result = ((2 + 3) * 4) - 1
print(result)  # 19
```

## Деление нацело

Оператор `//` выполняет деление и отбрасывает дробную часть (округляет вниз).

```python
print(17 // 5)   # 3 (17 / 5 = 3.4, целая часть = 3)
print(20 // 6)   # 3 (20 / 6 = 3.33..., целая часть = 3)
print(10 // 3)   # 3
print(7 // 7)    # 1
```

Деление нацело с отрицательными числами **округляет вниз**, а не к нулю:

```python
print(-7 // 2)   # -4 (не -3)
print(7 // -2)   # -4 (не -3)
```

### Практический пример

Сколько полных часов в 200 минутах?

```python
minutes = 200
hours = minutes // 60
remaining = minutes % 60
print(hours, "ч", remaining, "мин")  # 3 ч 20 мин
```

## Остаток от деления

Оператор `%` возвращает остаток от деления.

```python
print(10 % 3)   # 1  (10 = 3 * 3 + 1)
print(15 % 4)   # 3  (15 = 4 * 3 + 3)
print(20 % 5)   # 0  (20 делится нацело)
print(7 % 2)    # 1  (7 = 2 * 3 + 1)
```

### Частые применения оператора %

**Проверка чётности числа:**

```python
number = 42
if number % 2 == 0:
    print("Чётное")
else:
    print("Нечётное")
```

**Циклические индексы:** остаток всегда находится в диапазоне `[0, делитель - 1]`.

```python
# Дни недели: 0 = Пн, 6 = Вс
day = 10 % 7
print(day)  # 3 (четверг)
```

## Возведение в степень

Оператор `**` возводит число в степень.

```python
print(2 ** 10)   # 1024
print(3 ** 3)    # 27
print(5 ** 0)    # 1 (любое число в степени 0 = 1)
print(4 ** 0.5)  # 2.0 (квадратный корень)
print(8 ** (1/3))  # 2.0 (кубический корень)
```

## Операторы присваивания

Python предоставляет сокращённые операторы для изменения переменной:

| Оператор | Эквивалент | Описание |
|----------|------------|----------|
| `x += 5` | `x = x + 5` | Увеличить на 5 |
| `x -= 3` | `x = x - 3` | Уменьшить на 3 |
| `x *= 2` | `x = x * 2` | Умножить на 2 |
| `x /= 4` | `x = x / 4` | Разделить на 4 |
| `x //= 3` | `x = x // 3` | Разделить нацело на 3 |
| `x %= 7` | `x = x % 7` | Остаток от деления на 7 |
| `x **= 2` | `x = x ** 2` | Возвести в квадрат |

```python
score = 100
score += 10
print(score)  # 110

score -= 5
print(score)  # 105

score *= 2
print(score)  # 210
```

## Математические функции

Python имеет встроенные функции для базовых математических операций:

```python
print(abs(-5))     # 5  — абсолютное значение
print(round(3.7))  # 4  — округление
print(round(3.14159, 2))  # 3.14 — округление до 2 знаков
print(max(1, 5, 3))  # 5  — максимум
print(min(1, 5, 3))  # 1  — минимум
```

Для расширенных математических операций используйте модуль `math`:

```python
import math

print(math.sqrt(16))   # 4.0 — квадратный корень
print(math.pi)         # 3.141592653589793
print(math.floor(3.9)) # 3 — округление вниз
print(math.ceil(3.1))  # 4 — округление вверх
```
