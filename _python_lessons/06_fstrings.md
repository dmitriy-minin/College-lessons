## Способы форматирования строк

В Python существует несколько способов вставить переменные в строку:

```python
name = "Иван"
age = 25
```

| Способ | Код | Когда использовать |
|--------|-----|--------------------|
| Конкатенация | `"Привет, " + name` | Редко, для простых случаев |
| `%`-форматирование | `"Привет, %s" % name` | Устаревший стиль |
| `format()` | `"Привет, {}".format(name)` | Python 2.7+ |
| F-строки | `f"Привет, {name}"` | Современный стиль, Python 3.6+ |

Рекомендуется использовать **f-строки** — они самые читаемые и производительные.

---

## F-строки

F-строка — это строка с префиксом `f` или `F`. Внутри неё можно напрямую вставлять переменные и выражения в фигурных скобках `{}`.

```python
name = "Алексей"
age = 20

print(f"Привет, {name}!")           # Привет, Алексей!
print(f"Тебе {age} лет.")           # Тебе 20 лет.
print(f"{name} родился в {2024 - age} году.")  # Алексей родился в 2004 году.
```

F-строки работают и с другими типами данных:

```python
price = 99.99
count = 3

print(f"Цена: {price}")            # Цена: 99.99
print(f"Товаров: {count}")         # Товаров: 3
print(f"Итого: {price * count}")   # Итого: 299.97
```

## Выражения внутри f-строк

Внутри `{}` можно писать любые Python-выражения:

```python
a = 10
b = 3

print(f"Сумма: {a + b}")        # Сумма: 13
print(f"Произведение: {a * b}") # Произведение: 30
print(f"Степень: {a ** b}")     # Степень: 1000

name = "python"
print(f"Заглавные буквы: {name.upper()}")  # Заглавные буквы: PYTHON
print(f"Длина: {len(name)}")               # Длина: 6
```

Условные выражения:

```python
age = 17
status = "совершеннолетний" if age >= 18 else "несовершеннолетний"
print(f"Пользователь {status}.")  # Пользователь несовершеннолетний.
```

## Форматирование чисел

F-строки поддерживают мощный синтаксис форматирования после двоеточия: `{значение:формат}`.

### Числа с плавающей точкой

```python
pi = 3.14159265

print(f"{pi:.2f}")   # 3.14  (2 знака после точки)
print(f"{pi:.4f}")   # 3.1416 (4 знака)
print(f"{pi:.0f}")   # 3  (без дробной части)
```

### Ширина поля

```python
number = 42

print(f"{number:10}")    # "        42" (ширина 10, выравнивание вправо)
print(f"{number:<10}")   # "42        " (выравнивание влево)
print(f"{number:010}")   # "0000000042" (заполнение нулями)
```

### Разделители тысяч

```python
big_number = 1234567890

print(f"{big_number:,}")   # 1,234,567,890
print(f"{big_number:_}")   # 1_234_567_890
```

### Проценты

```python
ratio = 0.752

print(f"{ratio:.0%}")   # 75%
print(f"{ratio:.1%}")   # 75.2%
print(f"{ratio:.2%}")   # 75.20%
```

### Научная нотация

```python
value = 1234567.89

print(f"{value:e}")    # 1.234568e+06
print(f"{value:.2e}") # 1.23e+06
```

## Выравнивание текста

| Символ | Выравнивание | Пример |
|--------|-------------|--------|
| `<` | По левому краю | `f"{'текст':<10}"` |
| `>` | По правому краю | `f"{'текст':>10}"` |
| `^` | По центру | `f"{'текст':^10}"` |

```python
items = ["яблоко", "банан", "апельсин"]
prices = [15.5, 12.0, 30.75]

print(f"{'Товар':<12} {'Цена':>8}")
print("-" * 22)
for item, price in zip(items, prices):
    print(f"{item:<12} {price:>8.2f}")
```

Вывод:
```
Товар           Цена
----------------------
яблоко           15.50
банан            12.00
апельсин         30.75
```

Заполнение кастомным символом:

```python
title = "Отчёт"
print(f"{title:=^30}")  # ============Отчёт=============
print(f"{title:-^30}")  # ------------Отчёт-------------
```

## Метод format()

Старый, но всё ещё используемый способ форматирования. Полезно знать при чтении чужого кода.

### По позиции

```python
print("Привет, {}! Тебе {} лет.".format("Иван", 25))
# Привет, Иван! Тебе 25 лет.
```

### По индексу

```python
print("{0} и {1}, {1} и {0}".format("кофе", "чай"))
# кофе и чай, чай и кофе
```

### По имени

```python
print("{name} живёт в {city}.".format(name="Алексей", city="Москва"))
# Алексей живёт в Москве.
```

### Форматирование чисел через format()

```python
pi = 3.14159
print("{:.2f}".format(pi))  # 3.14
```
