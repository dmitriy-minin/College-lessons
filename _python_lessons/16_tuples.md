## Что такое кортеж

Кортеж (`tuple`) — это упорядоченная **неизменяемая** последовательность элементов. После создания кортежа нельзя добавить, удалить или изменить его элементы.

```python
coordinates = (55.75, 37.62)    # широта и долгота Москвы
rgb = (255, 128, 0)              # цвет в формате RGB
person = ("Иван", 25, "Москва") # имя, возраст, город
```

Кортежи используются для данных, которые не должны меняться.

---

## Создание кортежей

### Скобки

```python
empty = ()
single = (42,)          # обязательна запятая для кортежа из одного элемента
point = (3, 5)
colors = ("красный", "зелёный", "синий")
```

Обратите внимание: `(42)` — это просто число в скобках, а `(42,)` — кортеж.

```python
not_tuple = (42)
real_tuple = (42,)

print(type(not_tuple))    # <class 'int'>
print(type(real_tuple))   # <class 'tuple'>
```

### Без скобок

Python понимает кортеж даже без скобок, если есть запятые.

```python
point = 3, 5
name, age = "Иван", 25

print(point)   # (3, 5)
print(type(point))   # <class 'tuple'>
```

### Функция tuple()

```python
from_list = tuple([1, 2, 3])
print(from_list)   # (1, 2, 3)

from_string = tuple("Python")
print(from_string)   # ('P', 'y', 't', 'h', 'o', 'n')

from_range = tuple(range(5))
print(from_range)   # (0, 1, 2, 3, 4)
```

## Индексация и срезы

Работают точно так же, как у строк и списков.

```python
colors = ("красный", "зелёный", "синий", "жёлтый")

print(colors[0])     # красный
print(colors[-1])    # жёлтый
print(colors[1:3])   # ('зелёный', 'синий')
print(colors[::-1])  # ('жёлтый', 'синий', 'зелёный', 'красный')
```

Перебор в цикле:

```python
for color in colors:
    print(color)
```

## Неизменяемость кортежей

Нельзя изменить, добавить или удалить элемент.

```python
point = (3, 5)

point[0] = 10   # TypeError: объект tuple не поддерживает присваивание по индексу
```

Это ключевое свойство кортежа. Неизменяемость — не ограничение, а гарантия: данные точно останутся такими же.

Если нужен изменённый кортеж, создают новый:

```python
point = (3, 5)
new_point = (10,) + point[1:]
print(new_point)   # (10, 5)
```

### Конкатенация и умножение

```python
a = (1, 2)
b = (3, 4)

print(a + b)    # (1, 2, 3, 4)
print(a * 3)    # (1, 2, 1, 2, 1, 2)
```

## Методы кортежей

У кортежа только два метода — `count()` и `index()`.

### count() — количество вхождений

```python
numbers = (1, 2, 3, 2, 1, 2)
print(numbers.count(2))   # 3
print(numbers.count(5))   # 0
```

### index() — индекс первого вхождения

```python
colors = ("красный", "зелёный", "синий")
print(colors.index("зелёный"))   # 1
```

Остальные операции такие же, как у списков: `len()`, `min()`, `max()`, `sum()`, `in`.

```python
grades = (85, 92, 78, 91, 60)

print(len(grades))    # 5
print(max(grades))    # 92
print(min(grades))    # 60
print(sum(grades))    # 406
print(85 in grades)   # True
```

## Распаковка кортежей

Позволяет присвоить элементы кортежа переменным в одну строку.

```python
point = (3, 5)
x, y = point

print(x)   # 3
print(y)   # 5
```

Количество переменных должно совпадать с количеством элементов:

```python
person = ("Иван", 25, "Москва")
name, age, city = person

print(f"{name}, {age} лет, {city}")   # Иван, 25 лет, Москва
```

### Расширенная распаковка

Звёздочка `*` собирает оставшиеся элементы в список:

```python
numbers = (1, 2, 3, 4, 5)

first, *rest = numbers
print(first)   # 1
print(rest)    # [2, 3, 4, 5]

*start, last = numbers
print(start)   # [1, 2, 3, 4]
print(last)    # 5

first, *middle, last = numbers
print(first)    # 1
print(middle)   # [2, 3, 4]
print(last)     # 5
```

### Обмен значений переменных

```python
a = 10
b = 20

a, b = b, a   # под капотом создаётся кортеж (b, a)

print(a)   # 20
print(b)   # 10
```

## Кортеж vs список

| Свойство | Список `list` | Кортеж `tuple` |
|----------|--------------|----------------|
| Изменяемость | Да | Нет |
| Синтаксис | `[1, 2, 3]` | `(1, 2, 3)` |
| Методов | Много | 2 |
| Производительность | Чуть медленнее | Чуть быстрее |
| Использование как ключ словаря | Нет | Да |
| Когда использовать | Данные изменяются | Данные фиксированы |

### Когда использовать кортеж

- Координаты: `(x, y)`, `(lat, lon)`
- RGB-цвета: `(255, 0, 128)`
- Дата: `(2024, 1, 15)`
- Возврат нескольких значений из функции
- Ключ словаря (список не может быть ключом)

```python
# Кортеж как ключ словаря
distances = {
    ("Москва", "Питер"): 713,
    ("Москва", "Казань"): 797,
}
print(distances[("Москва", "Питер")])   # 713
```

## Практические примеры

### Возврат нескольких значений из функции

```python
def min_max(numbers):
    return min(numbers), max(numbers)

result = min_max([3, 1, 8, 5, 2])
minimum, maximum = result

print(f"Мин: {minimum}, Макс: {maximum}")   # Мин: 1, Макс: 8
```

### Фиксированные данные конфигурации

```python
ALLOWED_EXTENSIONS = (".jpg", ".jpeg", ".png", ".gif")
SERVER_ADDRESS = ("localhost", 8080)

host, port = SERVER_ADDRESS
print(f"Сервер запущен на {host}:{port}")
```

### Перебор пар данных

```python
months = (
    (1, "Январь", 31),
    (2, "Февраль", 28),
    (3, "Март", 31),
)

for number, name, days in months:
    print(f"{number:02d}. {name} — {days} дней")
```

Вывод:
```
01. Январь — 31 дней
02. Февраль — 28 дней
03. Март — 31 дней
```
