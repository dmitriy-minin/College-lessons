## Что такое переменная

Переменная — это именованная область памяти, в которой хранится значение. Думайте о переменной как о коробке с наклейкой: на наклейке написано имя, внутри — данные.

```python
name = "Алексей"   # создаём переменную name со значением "Алексей"
age = 20            # создаём переменную age со значением 20
```

Знак `=` — это **оператор присваивания**. Он не означает равенство, а записывает значение в переменную.

---

## Правила именования переменных

- Имя может содержать буквы, цифры и символ `_`
- Имя не может начинаться с цифры
- Имя чувствительно к регистру: `name` и `Name` — разные переменные
- Нельзя использовать зарезервированные слова: `if`, `for`, `while`, `print` и др.

```python
# Правильно
user_name = "Иван"
age2 = 25
_temp = 0

# Неправильно
2age = 25        # начинается с цифры
my-name = "Иван" # содержит дефис
for = 10         # зарезервированное слово
```

**Соглашение об именовании:** используйте `snake_case` — слова в нижнем регистре, разделённые символом подчёркивания.

```python
first_name = "Иван"
last_name = "Петров"
user_age = 30
```

## Тип int — целые числа

`int` хранит целые числа: положительные, отрицательные и ноль.

```python
score = 100
temperature = -15
year = 2024
zero = 0

print(score)        # 100
print(temperature)  # -15
```

Целые числа можно записывать с разделителями для удобочитаемости:

```python
population = 1_000_000   # то же самое, что 1000000
distance = 384_400       # расстояние до Луны в км
```

---

## Тип float — числа с плавающей точкой

`float` хранит числа с дробной частью. Разделитель дробной и целой части — точка.

```python
pi = 3.14159
price = 99.99
height = 1.75
negative = -0.5

print(pi)     # 3.14159
print(price)  # 99.99
```

Числа можно записывать в научной нотации:

```python
big = 1.5e6    # 1.5 * 10^6 = 1500000.0
small = 2.5e-3 # 2.5 * 10^-3 = 0.0025
```

> **Важно:** результат деления двух целых чисел всегда `float`.

```python
result = 10 / 2
print(result)  # 5.0, не 5
```

## Тип str — строки

`str` хранит текст. Строки заключаются в одинарные или двойные кавычки.

```python
first_name = "Алексей"
last_name = 'Иванов'
city = "Москва"
```

Многострочные строки записываются в тройных кавычках:

```python
description = """Это
многострочная
строка"""

print(description)
```

Вывод:
```
Это
многострочная
строка
```

Строка может быть пустой:

```python
empty = ""
print(empty)  # (пустая строка)
```

## Тип bool — логический тип

`bool` принимает только два значения: `True` (истина) или `False` (ложь). Обратите внимание — первая буква заглавная.

```python
is_adult = True
is_student = False
has_car = True

print(is_adult)    # True
print(is_student)  # False
```

Логический тип подробно рассматривается в уроке 7.

## Функция type()

Чтобы узнать тип переменной, используйте встроенную функцию `type()`.

```python
age = 25
price = 9.99
name = "Иван"
is_active = True

print(type(age))       # <class 'int'>
print(type(price))     # <class 'float'>
print(type(name))      # <class 'str'>
print(type(is_active)) # <class 'bool'>
```

## Преобразование типов

Python позволяет явно преобразовывать значения из одного типа в другой.

| Функция | Описание | Пример |
|---------|----------|--------|
| `int()` | В целое число | `int("42")` → `42` |
| `float()` | В число с точкой | `float("3.14")` → `3.14` |
| `str()` | В строку | `str(100)` → `"100"` |
| `bool()` | В логический тип | `bool(0)` → `False` |

```python
# Строку в число
age_str = "25"
age = int(age_str)
print(age + 5)  # 30

# Число в строку
score = 100
message = "Ваш счёт: " + str(score)
print(message)  # Ваш счёт: 100

# Целое в float
x = float(5)
print(x)  # 5.0
```

Некоторые преобразования невозможны:

```python
int("hello")  # ValueError: не удаётся преобразовать "hello" в число
```
