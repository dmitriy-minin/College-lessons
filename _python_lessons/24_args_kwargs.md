## Позиционные аргументы

Позиционные аргументы передаются в том же порядке, что и параметры в определении функции.

```python
def describe(name, age, city):
    print(f"{name}, {age} лет, {city}")

describe("Иван", 25, "Москва")     # Иван, 25 лет, Москва
describe("Мария", 30, "Питер")     # Мария, 30 лет, Питер
```

Порядок важен — если перепутать, результат будет неверным:

```python
describe(25, "Москва", "Иван")   # 25, Москва лет, Иван
```

## Именованные аргументы

Именованные (keyword) аргументы передаются с явным указанием имени параметра. Порядок при этом не важен.

```python
def describe(name, age, city):
    print(f"{name}, {age} лет, {city}")

describe(age=25, city="Москва", name="Иван")   # Иван, 25 лет, Москва
```

Можно смешивать: позиционные аргументы указываются первыми.

```python
describe("Иван", city="Москва", age=25)   # Иван, 25 лет, Москва
```

Именованные аргументы делают вызовы функций понятнее, особенно при нескольких параметрах одного типа:

```python
def create_rect(width, height, color="чёрный"):
    print(f"Прямоугольник {width}x{height}, цвет: {color}")

# Непонятно, что такое 100 и 200
create_rect(100, 200)

# Явно и читаемо
create_rect(width=100, height=200, color="красный")
```

## Только позиционные и только именованные

Python позволяет явно ограничить, каким способом передаются аргументы.

### Только именованные — после `*`

```python
def func(a, b, *, c, d):
    print(a, b, c, d)

func(1, 2, c=3, d=4)    # OK
func(1, 2, 3, 4)         # TypeError: c и d обязаны быть именованными
```

### Только позиционные — до `/`

```python
def func(a, b, /, c, d):
    print(a, b, c, d)

func(1, 2, 3, 4)              # OK
func(a=1, b=2, c=3, d=4)      # TypeError: a и b только позиционные
```

## *args — произвольное число позиционных аргументов

`*args` позволяет функции принять любое количество позиционных аргументов. Внутри функции они доступны как **кортеж**.

```python
def total(*args):
    print(args)       # кортеж всех аргументов
    return sum(args)

print(total(1, 2, 3))           # (1, 2, 3) → 6
print(total(10, 20, 30, 40))    # (10, 20, 30, 40) → 100
print(total())                  # () → 0
```

Имя `args` — только соглашение, технически можно назвать иначе, но менять не принято.

```python
def greet(*names):
    for name in names:
        print(f"Привет, {name}!")

greet("Иван")
greet("Мария", "Алексей", "Зина")
```

Вывод:
```
Привет, Иван!
Привет, Мария!
Привет, Алексей!
Привет, Зина!
```

Комбинация с обычными параметрами — `*args` идёт после фиксированных:

```python
def log(level, *messages):
    for msg in messages:
        print(f"[{level}] {msg}")

log("INFO", "Запуск сервера", "Порт 8080")
log("ERROR", "Подключение не удалось")
```

Вывод:
```
[INFO] Запуск сервера
[INFO] Порт 8080
[ERROR] Подключение не удалось
```

## **kwargs — произвольное число именованных аргументов

`**kwargs` позволяет принять любое количество именованных аргументов. Внутри функции они доступны как **словарь**.

```python
def describe(**kwargs):
    print(kwargs)
    for key, value in kwargs.items():
        print(f"{key}: {value}")

describe(name="Иван", age=25, city="Москва")
```

Вывод:
```
{'name': 'Иван', 'age': 25, 'city': 'Москва'}
name: Иван
age: 25
city: Москва
```

Имя `kwargs` — соглашение (keyword arguments), менять не рекомендуется.

```python
def create_user(**kwargs):
    # Дефолтные значения
    user = {"role": "user", "active": True}
    user.update(kwargs)
    return user

print(create_user(name="Иван", email="ivan@example.com"))
# {'role': 'user', 'active': True, 'name': 'Иван', 'email': 'ivan@example.com'}

print(create_user(name="Админ", role="admin"))
# {'role': 'admin', 'active': True, 'name': 'Админ'}
```

## Комбинирование всех видов аргументов

Все виды аргументов можно использовать вместе. Порядок в определении обязателен:

```
def func(позиционные, *args, именованные_с_дефолтом, **kwargs)
```

```python
def full_function(a, b, *args, keyword=None, **kwargs):
    print(f"a={a}, b={b}")
    print(f"args={args}")
    print(f"keyword={keyword}")
    print(f"kwargs={kwargs}")

full_function(1, 2, 3, 4, keyword="привет", x=10, y=20)
```

Вывод:
```
a=1, b=2
args=(3, 4)
keyword=привет
kwargs={'x': 10, 'y': 20}
```

## Распаковка при вызове функции

`*` и `**` можно использовать не только в определении, но и при **вызове** функции для распаковки коллекций.

### Распаковка списка/кортежа через *

```python
def add(a, b, c):
    return a + b + c

numbers = [1, 2, 3]
print(add(*numbers))   # 6  — равнозначно add(1, 2, 3)

coords = (10, 20, 30)
print(add(*coords))    # 60
```

### Распаковка словаря через **

```python
def describe(name, age, city):
    print(f"{name}, {age}, {city}")

person = {"name": "Иван", "age": 25, "city": "Москва"}
describe(**person)   # Иван, 25, Москва
```

### Объединение списков и словарей

```python
list1 = [1, 2, 3]
list2 = [4, 5, 6]
combined = [*list1, *list2]
print(combined)   # [1, 2, 3, 4, 5, 6]

dict1 = {"a": 1, "b": 2}
dict2 = {"c": 3, "d": 4}
merged = {**dict1, **dict2}
print(merged)   # {'a': 1, 'b': 2, 'c': 3, 'd': 4}
```

## Практические примеры

### Гибкая функция форматирования

```python
def format_message(template, *args, **kwargs):
    """Форматирует шаблон с позиционными или именованными аргументами."""
    if args:
        return template.format(*args)
    return template.format(**kwargs)

print(format_message("Привет, {}! Тебе {} лет.", "Иван", 25))
print(format_message("Привет, {name}! Город: {city}.", name="Мария", city="Питер"))
```

### Логгер с произвольными данными

```python
def log(level, message, **context):
    parts = [f"[{level}] {message}"]
    for k, v in context.items():
        parts.append(f"{k}={v}")
    print(" | ".join(parts))

log("INFO", "Пользователь вошёл", user_id=42, ip="192.168.1.1")
log("ERROR", "Ошибка БД", query="SELECT *", error_code=500)
```

Вывод:
```
[INFO] Пользователь вошёл | user_id=42 | ip=192.168.1.1
[ERROR] Ошибка БД | query=SELECT * | error_code=500
```
