## Обзор методов

| Метод | Описание |
|-------|----------|
| `append(x)` | Добавляет элемент `x` в конец |
| `extend(iter)` | Добавляет все элементы из `iter` в конец |
| `insert(i, x)` | Вставляет элемент `x` перед индексом `i` |
| `remove(x)` | Удаляет первое вхождение `x` |
| `pop(i)` | Удаляет и возвращает элемент с индексом `i` |
| `clear()` | Очищает список |
| `index(x)` | Возвращает индекс первого вхождения `x` |
| `count(x)` | Считает количество вхождений `x` |
| `reverse()` | Разворачивает список на месте |
| `copy()` | Возвращает поверхностную копию |

## append() — добавить в конец

Добавляет один элемент в конец списка. Изменяет список на месте, ничего не возвращает.

```python
fruits = ["яблоко", "банан"]

fruits.append("вишня")
print(fruits)   # ['яблоко', 'банан', 'вишня']

fruits.append("манго")
print(fruits)   # ['яблоко', 'банан', 'вишня', 'манго']
```

Если передать список, он добавится как один элемент:

```python
items = [1, 2, 3]
items.append([4, 5])
print(items)    # [1, 2, 3, [4, 5]]  — вложенный список!
```

Типичное применение — сборка списка в цикле:

```python
squares = []

for i in range(1, 6):
    squares.append(i ** 2)

print(squares)  # [1, 4, 9, 16, 25]
```

## extend() — расширить список

Добавляет все элементы из другой последовательности. В отличие от `append`, разворачивает переданную коллекцию.

```python
numbers = [1, 2, 3]
numbers.extend([4, 5, 6])
print(numbers)   # [1, 2, 3, 4, 5, 6]
```

Разница между `append` и `extend`:

```python
a = [1, 2, 3]
b = [1, 2, 3]

a.append([4, 5])    # добавляет список как элемент
b.extend([4, 5])    # добавляет элементы списка

print(a)   # [1, 2, 3, [4, 5]]
print(b)   # [1, 2, 3, 4, 5]
```

Можно расширять строкой или `range`:

```python
letters = ["a", "b"]
letters.extend("cde")
print(letters)   # ['a', 'b', 'c', 'd', 'e']

nums = [1, 2]
nums.extend(range(3, 6))
print(nums)      # [1, 2, 3, 4, 5]
```

## insert() — вставить по индексу

Вставляет элемент **перед** указанным индексом.

```python
insert(index, element)
```

```python
fruits = ["яблоко", "вишня", "манго"]

fruits.insert(1, "банан")   # вставить перед индексом 1
print(fruits)   # ['яблоко', 'банан', 'вишня', 'манго']
```

Особые случаи:

```python
items = [1, 2, 3]

items.insert(0, 0)      # вставить в начало
print(items)   # [0, 1, 2, 3]

items.insert(100, 99)   # индекс больше длины — вставка в конец
print(items)   # [0, 1, 2, 3, 99]
```

## remove() — удалить по значению

Удаляет **первое вхождение** указанного значения. Если элемент не найден — вызывает `ValueError`.

```python
fruits = ["яблоко", "банан", "вишня", "банан"]

fruits.remove("банан")
print(fruits)   # ['яблоко', 'вишня', 'банан']  — удалено только первое
```

Безопасное удаление через проверку:

```python
fruits = ["яблоко", "вишня"]

item_to_remove = "банан"
if item_to_remove in fruits:
    fruits.remove(item_to_remove)
else:
    print(f"'{item_to_remove}' не найдено в списке")
```

## pop() — извлечь элемент

Удаляет элемент по индексу и **возвращает его**. Без аргумента — удаляет последний элемент.

```python
fruits = ["яблоко", "банан", "вишня", "манго"]

last = fruits.pop()
print(last)      # манго
print(fruits)    # ['яблоко', 'банан', 'вишня']

second = fruits.pop(1)
print(second)    # банан
print(fruits)    # ['яблоко', 'вишня']
```

`pop()` — единственный метод, который одновременно удаляет и возвращает значение. Это полезно при реализации стека.

### Стек (LIFO) через список

```python
stack = []

stack.append("первый")
stack.append("второй")
stack.append("третий")

print(stack.pop())   # третий
print(stack.pop())   # второй
print(stack.pop())   # первый
```

## Дополнительные методы

### clear() — очистить список

```python
items = [1, 2, 3, 4, 5]
items.clear()
print(items)   # []
```

### index() — найти индекс элемента

```python
fruits = ["яблоко", "банан", "вишня"]
print(fruits.index("банан"))    # 1
print(fruits.index("яблоко"))   # 0

# Поиск в диапазоне: index(x, start, end)
numbers = [1, 2, 3, 2, 5]
print(numbers.index(2, 2))   # 3  — ищет начиная с индекса 2
```

### count() — количество вхождений

```python
numbers = [1, 2, 3, 2, 1, 2]
print(numbers.count(2))   # 3
print(numbers.count(5))   # 0
```

### copy() — копия списка

```python
original = [1, 2, 3]
copy = original.copy()

copy.append(4)
print(original)   # [1, 2, 3]  — не изменился
print(copy)       # [1, 2, 3, 4]
```

Зачем нужна копия — без неё оба имени указывают на один объект:

```python
a = [1, 2, 3]
b = a           # не копия — оба смотрят на один список

b.append(4)
print(a)        # [1, 2, 3, 4]  — изменился!
```

## Сравнение методов удаления

| Метод | По чему удаляет | Возвращает | Ошибка если нет |
|-------|----------------|-----------|-----------------|
| `remove(x)` | По значению | `None` | `ValueError` |
| `pop(i)` | По индексу | Удалённый элемент | `IndexError` |
| `del list[i]` | По индексу | — | `IndexError` |
| `clear()` | — (всё) | `None` | Нет |

## Практические примеры

### Очередь (FIFO)

```python
queue = []

queue.append("Иван")
queue.append("Мария")
queue.append("Алексей")

print(f"Обслуживается: {queue.pop(0)}")   # Иван
print(f"Обслуживается: {queue.pop(0)}")   # Мария
print(f"В очереди: {queue}")              # ['Алексей']
```

### Фильтрация списка на месте

```python
scores = [45, 82, 60, 91, 55, 78, 40, 95]
threshold = 60

i = 0
while i < len(scores):
    if scores[i] < threshold:
        scores.pop(i)
    else:
        i += 1

print(scores)   # [82, 60, 91, 78, 95]
```

### Сбор уникальных значений

```python
raw_data = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3]
unique = []

for item in raw_data:
    if item not in unique:
        unique.append(item)

print(unique)   # [3, 1, 4, 5, 9, 2, 6]
```
