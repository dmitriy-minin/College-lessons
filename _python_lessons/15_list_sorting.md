## sort() — сортировка на месте

Метод `sort()` сортирует список **на месте** — изменяет исходный список и возвращает `None`.

```python
numbers = [3, 1, 4, 1, 5, 9, 2, 6]

numbers.sort()
print(numbers)   # [1, 1, 2, 3, 4, 5, 6, 9]
```

Строки сортируются лексикографически (по алфавиту):

```python
fruits = ["банан", "яблоко", "вишня", "апельсин"]

fruits.sort()
print(fruits)   # ['апельсин', 'банан', 'вишня', 'яблоко']
```

Важно: нельзя смешивать несовместимые типы.

```python
mixed = [3, "text", 1]
mixed.sort()   # TypeError: не удаётся сравнить str и int
```

## sorted() — новый отсортированный список

Функция `sorted()` возвращает **новый отсортированный список**, исходный не трогает.

```python
numbers = [3, 1, 4, 1, 5, 9, 2, 6]

result = sorted(numbers)
print(result)    # [1, 1, 2, 3, 4, 5, 6, 9]
print(numbers)   # [3, 1, 4, 1, 5, 9, 2, 6]  — не изменился
```

`sorted()` работает с любой последовательностью, не только со списками:

```python
word = "python"
print(sorted(word))   # ['h', 'n', 'o', 'p', 't', 'y']

nums = (5, 3, 8, 1)   # кортеж
print(sorted(nums))    # [1, 3, 5, 8]  — возвращает список
```

## sort() vs sorted()

| | `sort()` | `sorted()` |
|-|----------|------------|
| Тип | Метод списка | Встроенная функция |
| Возвращает | `None` | Новый список |
| Изменяет оригинал | Да | Нет |
| Работает с | Только списки | Любые последовательности |
| Когда использовать | Оригинал не нужен | Нужно сохранить оригинал |

```python
# sort() — изменяет список, возвращает None
data = [3, 1, 2]
result = data.sort()
print(result)   # None
print(data)     # [1, 2, 3]

# sorted() — сохраняет оригинал, возвращает новый список
data = [3, 1, 2]
result = sorted(data)
print(result)   # [1, 2, 3]
print(data)     # [3, 1, 2]
```

## Параметр reverse

Оба метода поддерживают параметр `reverse=True` для сортировки по убыванию.

```python
numbers = [3, 1, 4, 1, 5, 9]

numbers.sort(reverse=True)
print(numbers)   # [9, 5, 4, 3, 1, 1]

words = ["банан", "яблоко", "вишня"]
print(sorted(words, reverse=True))   # ['яблоко', 'вишня', 'банан']
```

## Параметр key — сортировка по критерию

Параметр `key` задаёт функцию, которая вычисляет значение для сравнения.

### Сортировка по длине строки

```python
words = ["банан", "яблоко", "кот", "апельсин", "мак"]

words.sort(key=len)
print(words)   # ['кот', 'мак', 'банан', 'яблоко', 'апельсин']
```

### Сортировка без учёта регистра

```python
names = ["Зина", "антон", "Михаил", "борис"]

names.sort(key=str.lower)
print(names)   # ['антон', 'борис', 'Зина', 'Михаил']
```

### Сортировка по последнему символу

```python
words = ["Python", "Java", "C", "Go", "Rust"]

print(sorted(words, key=lambda w: w[-1]))
# ['Java', 'C', 'Rust', 'Python', 'Go']
```

### Сортировка словарей по полю

```python
students = [
    {"name": "Иван", "grade": 85},
    {"name": "Мария", "grade": 92},
    {"name": "Алексей", "grade": 78},
]

students.sort(key=lambda s: s["grade"])
for s in students:
    print(f"{s['name']}: {s['grade']}")
```

Вывод:
```
Алексей: 78
Иван: 85
Мария: 92
```

### Сортировка по нескольким критериям

```python
data = [
    ("Иван", 25),
    ("Мария", 30),
    ("Алексей", 25),
    ("Зина", 30),
]

# Сначала по возрасту, затем по имени
data.sort(key=lambda x: (x[1], x[0]))
print(data)
# [('Алексей', 25), ('Иван', 25), ('Зина', 30), ('Мария', 30)]
```

## Разворот списка

### reverse() — разворот на месте

Метод `reverse()` переворачивает список **на месте** и возвращает `None`.

```python
numbers = [1, 2, 3, 4, 5]

numbers.reverse()
print(numbers)   # [5, 4, 3, 2, 1]
```

### Срез [::-1] — новый развёрнутый список

```python
numbers = [1, 2, 3, 4, 5]

reversed_numbers = numbers[::-1]
print(reversed_numbers)   # [5, 4, 3, 2, 1]
print(numbers)             # [1, 2, 3, 4, 5]  — оригинал цел
```

### reversed() — итератор в обратном порядке

```python
numbers = [1, 2, 3, 4, 5]

for num in reversed(numbers):
    print(num, end=" ")
# 5 4 3 2 1
```

### Сравнение методов разворота

| Способ | Изменяет оригинал | Возвращает |
|--------|-------------------|-----------|
| `reverse()` | Да | `None` |
| `[::-1]` | Нет | Новый список |
| `reversed()` | Нет | Итератор |

## Практические примеры

### Топ-3 результата

```python
scores = [45, 82, 60, 91, 55, 78, 40, 95]

top3 = sorted(scores, reverse=True)[:3]
print(f"Топ-3: {top3}")   # Топ-3: [95, 91, 82]
```

### Сортировка файлов по расширению

```python
files = ["report.pdf", "image.png", "script.py", "data.csv", "main.py"]

files.sort(key=lambda f: f.split(".")[-1])
print(files)
# ['data.csv', 'report.pdf', 'image.png', 'main.py', 'script.py']
```

### Медиана списка

```python
numbers = [7, 2, 10, 9, 1]
sorted_nums = sorted(numbers)
n = len(sorted_nums)

if n % 2 == 1:
    median = sorted_nums[n // 2]
else:
    median = (sorted_nums[n // 2 - 1] + sorted_nums[n // 2]) / 2

print(f"Медиана: {median}")   # 7
```
