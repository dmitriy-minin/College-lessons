## Что такое словарь

Словарь (`dict`) — это коллекция пар **ключ: значение**. Ключ уникален, значение может быть любым. Словари неупорядочены в теории, но на практике в Python 3.7+ сохраняют порядок вставки.

```python
person = {
    "name": "Иван",
    "age": 25,
    "city": "Москва"
}
```

По сравнению со списком, словарь даёт доступ к данным **по имени**, а не по номеру. Это делает код понятнее.

```python
# Список — нужно помнить порядок
person_list = ["Иван", 25, "Москва"]
print(person_list[1])   # 25 — что это?

# Словарь — имена говорят сами за себя
person_dict = {"name": "Иван", "age": 25, "city": "Москва"}
print(person_dict["age"])   # 25 — понятно!
```

## Создание словарей

### Фигурные скобки

```python
empty = {}

capitals = {
    "Россия": "Москва",
    "Франция": "Париж",
    "Германия": "Берлин",
}

scores = {"Иван": 85, "Мария": 92, "Алексей": 78}
```

### Функция dict()

```python
person = dict(name="Иван", age=25, city="Москва")
print(person)   # {'name': 'Иван', 'age': 25, 'city': 'Москва'}
```

### Из списка пар

```python
pairs = [("a", 1), ("b", 2), ("c", 3)]
d = dict(pairs)
print(d)   # {'a': 1, 'b': 2, 'c': 3}
```

### dict.fromkeys()

Создаёт словарь с одинаковым значением для всех ключей.

```python
keys = ["x", "y", "z"]
d = dict.fromkeys(keys, 0)
print(d)   # {'x': 0, 'y': 0, 'z': 0}
```

## Доступ к значениям

### По ключу через []

```python
capitals = {"Россия": "Москва", "Франция": "Париж"}

print(capitals["Россия"])    # Москва
print(capitals["Франция"])   # Париж
```

Обращение к несуществующему ключу вызывает `KeyError`:

```python
print(capitals["Испания"])   # KeyError: 'Испания'
```

### Метод get() — безопасный доступ

Подробнее рассматривается в следующем уроке. Краткий обзор:

```python
print(capitals.get("Испания"))           # None (без ошибки)
print(capitals.get("Испания", "Неизвестно"))  # Неизвестно
```

## Добавление и изменение элементов

Словарь изменяемый. Новые пары добавляются присваиванием по ключу:

```python
person = {"name": "Иван", "age": 25}

# Добавление нового ключа
person["city"] = "Москва"
print(person)   # {'name': 'Иван', 'age': 25, 'city': 'Москва'}

# Изменение существующего значения
person["age"] = 26
print(person)   # {'name': 'Иван', 'age': 26, 'city': 'Москва'}
```

Если ключ есть — значение заменяется. Если нет — создаётся новая пара.

### Метод update()

Обновляет словарь несколькими парами сразу:

```python
person = {"name": "Иван", "age": 25}
person.update({"age": 26, "city": "Питер", "job": "программист"})
print(person)
# {'name': 'Иван', 'age': 26, 'city': 'Питер', 'job': 'программист'}
```

## Удаление элементов

### del

```python
person = {"name": "Иван", "age": 25, "city": "Москва"}

del person["city"]
print(person)   # {'name': 'Иван', 'age': 25}
```

### pop()

Удаляет ключ и возвращает значение. Можно указать значение по умолчанию:

```python
person = {"name": "Иван", "age": 25}

age = person.pop("age")
print(age)      # 25
print(person)   # {'name': 'Иван'}

# С дефолтным значением — нет ошибки при отсутствии ключа
value = person.pop("city", "не указан")
print(value)   # не указан
```

### clear()

```python
data = {"a": 1, "b": 2}
data.clear()
print(data)   # {}
```

## Перебор словаря

### По ключам (по умолчанию)

```python
capitals = {"Россия": "Москва", "Франция": "Париж", "Германия": "Берлин"}

for country in capitals:
    print(country)
# Россия
# Франция
# Германия
```

### По значениям

```python
for capital in capitals.values():
    print(capital)
# Москва
# Париж
# Берлин
```

### По парам ключ-значение

```python
for country, capital in capitals.items():
    print(f"Столица {country}: {capital}")
# Столица России: Москва
# Столица Франции: Париж
# Столица Германии: Берлин
```

## Проверка наличия ключа

Оператор `in` проверяет наличие **ключа** (не значения).

```python
person = {"name": "Иван", "age": 25}

print("name" in person)     # True
print("city" in person)     # False
print("city" not in person) # True
```

## Вложенные словари

Значением может быть другой словарь.

```python
students = {
    "Иван": {"grade": 9, "score": 85},
    "Мария": {"grade": 10, "score": 92},
    "Алексей": {"grade": 9, "score": 78},
}

print(students["Мария"]["score"])   # 92

# Перебор вложенного словаря
for name, info in students.items():
    print(f"{name}: класс {info['grade']}, балл {info['score']}")
```

Вывод:
```
Иван: класс 9, балл 85
Мария: класс 10, балл 92
Алексей: класс 9, балл 78
```
