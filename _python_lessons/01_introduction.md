## Что такое Python

Python — высокоуровневый язык программирования с простым и читаемым синтаксисом. Он широко используется в веб-разработке, анализе данных, автоматизации, машинном обучении и многих других областях.

Ключевые особенности Python:

- Простой синтаксис, близкий к естественному языку
- Интерпретируемый язык — код выполняется строка за строкой
- Кроссплатформенный — работает на Windows, macOS, Linux
- Огромная стандартная библиотека и экосистема пакетов

---

## Установка Python

### Windows

1. Перейдите на официальный сайт: [python.org/downloads](https://www.python.org/downloads/)
2. Скачайте последнюю стабильную версию Python 3.x
3. Запустите установщик
4. **Обязательно** поставьте галочку `Add Python to PATH`
5. Нажмите `Install Now`

### macOS

```bash
# Через Homebrew (рекомендуется)
brew install python3
```

Либо скачайте установщик с [python.org](https://www.python.org/downloads/).

### Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install python3 python3-pip
```

### Проверка установки

После установки откройте терминал и выполните:

```bash
python --version
# или
python3 --version
```

Вы должны увидеть что-то вроде: `Python 3.12.0`

## Выбор редактора кода

Для написания Python-кода рекомендуются два редактора:

| Редактор | Преимущества | Для кого |
|----------|-------------|----------|
| **PyCharm** | Полноценная IDE, автодополнение, отладчик | Новичкам и профессионалам |
| **VS Code** | Лёгкий, гибкий, множество расширений | Тем, кто хочет универсальный инструмент |

---

## Настройка PyCharm

1. Скачайте **PyCharm Community Edition** (бесплатно) с [jetbrains.com/pycharm](https://www.jetbrains.com/pycharm/)
2. Установите и запустите
3. Создайте новый проект: `File → New Project`
4. Укажите папку проекта и выберите интерпретатор Python
5. Нажмите `Create`

---

## Настройка VS Code

1. Скачайте с [code.visualstudio.com](https://code.visualstudio.com/)
2. Установите расширение **Python** от Microsoft:
   - Нажмите `Ctrl+Shift+X`
   - Найдите `Python`
   - Установите расширение от Microsoft
3. Откройте папку проекта: `File → Open Folder`
4. Создайте файл с расширением `.py`

## Первая программа

Создайте файл `hello.py` и напишите в нём:

```python
print("Hello, World!")
```

Разберём эту строку:

- `print` — встроенная функция Python для вывода текста
- `"Hello, World!"` — строка (текст) в кавычках
- Скобки `()` — вызов функции

Усложним программу:

```python
print("Hello, World!")
print("Меня зовут Python")
print("Мне", 33, "года")
```

---

## Запуск программы

### Через терминал

```bash
python hello.py
# или
python3 hello.py
```

### В PyCharm

Нажмите зелёную кнопку `Run` (треугольник) или `Shift+F10`.

### В VS Code

Нажмите кнопку `Run` в правом верхнем углу или `Ctrl+F5`.

### Ожидаемый вывод

```
Hello, World!
Меня зовут Python
Мне 33 года
```
