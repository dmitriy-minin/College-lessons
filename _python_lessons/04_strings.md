## Строки в Python

Строка (`str`) — это последовательность символов. Строки можно создавать несколькими способами:

```python
s1 = "Привет"         # двойные кавычки
s2 = 'Мир'            # одинарные кавычки
s3 = """Многострочная
строка"""              # тройные кавычки

print(s1)  # Привет
print(s2)  # Мир
print(s3)
# Многострочная
# строка
```

### Специальные символы

| Символ | Значение |
|--------|----------|
| `\n` | Перенос строки |
| `\t` | Табуляция |
| `\\` | Обратный слеш |
| `\"` | Двойная кавычка внутри строки |
| `\'` | Одинарная кавычка внутри строки |

```python
print("Первая строка\nВторая строка")
# Первая строка
# Вторая строка

print("Колонка1\tКолонка2")
# Колонка1    Колонка2
```

## Конкатенация строк

Конкатенация — это объединение строк. В Python для этого используется оператор `+`.

```python
first = "Иван"
last = "Петров"

full_name = first + " " + last
print(full_name)  # Иван Петров
```

Можно объединять несколько строк:

```python
greeting = "Привет" + ", " + "мир" + "!"
print(greeting)  # Привет, мир!
```

Нельзя напрямую конкатенировать строку и число — нужно преобразование:

```python
age = 25
# print("Мне " + age + " лет")  # TypeError!

print("Мне " + str(age) + " лет")  # Правильно: Мне 25 лет
```

### Оператор += для строк

```python
message = "Привет"
message += ", мир"
message += "!"
print(message)  # Привет, мир!
```

## Дублирование строк

Оператор `*` повторяет строку указанное количество раз.

```python
line = "-" * 30
print(line)  # ------------------------------

word = "Ха"
laugh = word * 3
print(laugh)  # ХаХаХа

separator = "=-" * 10
print(separator)  # =-=-=-=-=-=-=-=-=-=-
```

Дублирование полезно для создания визуальных разделителей:

```python
title = "Отчёт"
print("=" * 20)
print(title)
print("=" * 20)
# ====================
# Отчёт
# ====================
```

## Длина строки — len()

Функция `len()` возвращает количество символов в строке, включая пробелы.

```python
name = "Александр"
print(len(name))  # 9

with_spaces = "Hello World"
print(len(with_spaces))  # 11 (пробел тоже считается)

empty = ""
print(len(empty))  # 0
```

### Практическое применение

Проверка минимальной длины пароля:

```python
password = "mypassword123"
min_length = 8

if len(password) >= min_length:
    print("Пароль достаточно длинный")
else:
    print("Пароль слишком короткий")
```

Ограничение длины текста:

```python
tweet = "Это мой первый твит в Python!"
max_chars = 280

print("Длина:", len(tweet))
print("Осталось символов:", max_chars - len(tweet))
```

## Индексация

Каждый символ строки имеет свой индекс. Нумерация начинается с `0`.

```python
text = "Python"
#       P y t h o n
#       0 1 2 3 4 5   (прямая индексация)
#      -6-5-4-3-2-1   (обратная индексация)
```

Доступ к символу по индексу:

```python
text = "Python"

print(text[0])   # P
print(text[1])   # y
print(text[5])   # n
print(text[-1])  # n (последний символ)
print(text[-2])  # o (предпоследний)
```

Обращение к несуществующему индексу вызывает ошибку:

```python
text = "Python"
print(text[10])  # IndexError: string index out of range
```

## Срезы

Срез позволяет получить подстроку по диапазону индексов.

Синтаксис: `строка[начало:конец:шаг]`

- `начало` — включается
- `конец` — не включается
- `шаг` — по умолчанию 1

```python
text = "Python"

print(text[0:3])   # Pyt  (с 0 по 2 включительно)
print(text[2:5])   # tho  (с 2 по 4)
print(text[3:])    # hon  (с 3 до конца)
print(text[:4])    # Pyth (с начала до 3)
print(text[:])     # Python (вся строка)
print(text[::2])   # Pto  (каждый второй символ)
print(text[::-1])  # nohtyP (строка наоборот)
```

Срезы с отрицательными индексами:

```python
text = "Hello, World"

print(text[-5:])    # World
print(text[-5:-1])  # Worl
```

## Неизменяемость строк

Строки в Python **неизменяемые** — нельзя изменить отдельный символ.

```python
text = "Python"
text[0] = "J"  # TypeError: строки не поддерживают присваивание по индексу
```

Чтобы изменить строку, нужно создать новую:

```python
text = "Python"
text = "J" + text[1:]
print(text)  # Jython
```
