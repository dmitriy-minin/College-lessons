### 1. Что такое Python?

**Python** (Пайтон) — это `универсальный язык` программирования высокого уровня. Он был создан в конце 80-х годов **Гвидо ван Россумом** и назван не в честь змеи, а в честь британского комедийного шоу «Монти Пайтон».

Главная философия языка заложена в документе под названием **The Zen of Python**. Основные принципы:
* Красивое лучше, чем уродливое.
* Явное лучше, чем неявное.
* Простое лучше, чем сложное.
* Читаемость имеет значение.

---

### 2. Почему Python так популярен?

На сегодняшний день Python занимает первые строчки в рейтингах популярности (TIOBE, GitHub). Вот основные причины:

1.  **Лаконичность:** То, что в Java или C++ пишется в 10 строк, в Python можно решить одной.
2.  **Огромная экосистема:** Существует более 300 000 библиотек (пакетов) для любых задач.
3.  **Интерпретируемость:** Код выполняется построчно, что позволяет быстро тестировать идеи без долгой компиляции.
4.  **Кроссплатформенность:** Код, написанный на Windows, будет работать на Linux (Ubuntu) и macOS без изменений.

---

### 3. Где применяется Python?

* **Web-разработка:** Мощные фреймворки, такие как **Django** и **Flask**, позволяют создавать масштабируемые сайты (Instagram, Pinterest и Spotify написаны на Python).
* **Data Science и ИИ:** Лидер в области машинного обучения и анализа данных (библиотеки Pandas, NumPy, TensorFlow).
* **Автоматизация:** Написание скриптов для работы с файлами, парсинга сайтов и автоматизации рутинных задач.
* **Разработка ботов:** Создание чат-ботов для Telegram, Discord и других платформ.

### 4. Настройка рабочего окружения

Для полноценной работы нам понадобятся два инструмента:

1.  **Интерпретатор Python:** Рекомендуется версия **3.10** или выше.
    * *В Ubuntu установка через терминал:* `sudo apt update && sudo apt install python3`
2.  **Редактор кода (IDE):** * **VS Code** (универсальный).
    * **PyCharm** (профессиональный для Python).
