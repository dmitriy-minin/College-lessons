## Оператор break

`break` немедленно прерывает выполнение цикла и передаёт управление коду после него.

```python
for i in range(10):
    if i == 5:
        break
    print(i)

print("После цикла")
```

Вывод:
```
0
1
2
3
4
После цикла
```

Как только `i` стало равно `5`, цикл прервался — числа `5, 6, 7, 8, 9` не вывелись.

### break в while

```python
while True:
    number = int(input("Введите положительное число (0 — выход): "))
    if number == 0:
        break
    print(f"Вы ввели: {number}")

print("Программа завершена")
```

### Поиск элемента

```python
names = ["Анна", "Борис", "Виктор", "Галина"]
target = "Виктор"

for name in names:
    if name == target:
        print(f"Найдено: {name}")
        break
```

### break во вложенных циклах

Важно: `break` прерывает только **ближайший** цикл.

```python
for i in range(3):
    for j in range(3):
        if j == 1:
            break          # прерывает только внутренний цикл
        print(f"i={i}, j={j}")
```

Вывод:
```
i=0, j=0
i=1, j=0
i=2, j=0
```

## Оператор continue

`continue` пропускает оставшуюся часть текущей итерации и переходит к следующей. Цикл при этом не прерывается.

```python
for i in range(10):
    if i % 2 == 0:
        continue       # пропустить чётные
    print(i)
```

Вывод:
```
1
3
5
7
9
```

### continue в while

```python
n = 0

while n < 10:
    n += 1
    if n % 3 == 0:
        continue       # пропустить кратные 3
    print(n)
```

Вывод:
```
1
2
4
5
7
8
10
```

### Отличие break от continue

| Оператор | Действие |
|----------|----------|
| `break` | Завершает весь цикл |
| `continue` | Пропускает текущую итерацию, цикл продолжается |

```python
# break — останавливается на 5
for i in range(1, 10):
    if i == 5:
        break
    print(i)
# 1 2 3 4

# continue — пропускает 5
for i in range(1, 10):
    if i == 5:
        continue
    print(i)
# 1 2 3 4 6 7 8 9
```

## Блок else в цикле

Python позволяет добавить блок `else` к циклу. Он выполняется, если цикл завершился **без вызова** `break`.

```python
for переменная in последовательность:
    # тело цикла
else:
    # выполняется, если цикл не был прерван через break
```

```python
for i in range(5):
    print(i)
else:
    print("Цикл завершён нормально")
```

Вывод:
```
0
1
2
3
4
Цикл завершён нормально
```

Если `break` сработал — `else` не выполняется:

```python
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("Цикл завершён нормально")  # НЕ выполнится
```

Вывод:
```
0
1
2
```

### else как индикатор "не найдено"

Это самый распространённый сценарий использования `else` в циклах.

```python
numbers = [4, 7, 12, 9, 3]
target = 15

for num in numbers:
    if num == target:
        print(f"Число {target} найдено!")
        break
else:
    print(f"Число {target} не найдено.")
```

Вывод:
```
Число 15 не найдено.
```

### else с while

```python
attempts = 0
password = "secret"

while attempts < 3:
    entered = input("Введите пароль: ")
    attempts += 1
    if entered == password:
        print("Доступ разрешён")
        break
else:
    print("Превышено количество попыток. Доступ заблокирован.")
```

## Комбинирование операторов

`break` и `continue` можно использовать вместе в одном цикле.

```python
for i in range(1, 20):
    if i % 2 == 0:
        continue       # пропускаем чётные
    if i > 15:
        break          # останавливаемся после 15
    print(i)
```

Вывод:
```
1
3
5
7
9
11
13
15
```

## Практические примеры

### Проверка простого числа

```python
number = 29
is_prime = True

if number < 2:
    is_prime = False
else:
    for i in range(2, int(number ** 0.5) + 1):
        if number % i == 0:
            is_prime = False
            break

if is_prime:
    print(f"{number} — простое число")
else:
    print(f"{number} — составное число")
```

### Фильтрация данных

```python
data = [15, -3, 42, -7, 0, 28, -1, 100]
positive_sum = 0

for value in data:
    if value <= 0:
        continue
    positive_sum += value

print(f"Сумма положительных: {positive_sum}")  # 185
```

### Поиск первого совпадения

```python
users = [
    {"name": "Иван", "age": 25},
    {"name": "Мария", "age": 17},
    {"name": "Алексей", "age": 30},
]

for user in users:
    if user["age"] < 18:
        print(f"Первый несовершеннолетний: {user['name']}")
        break
else:
    print("Все пользователи совершеннолетние")
```
