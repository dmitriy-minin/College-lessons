## Зачем нужны условия

Программа должна уметь принимать решения в зависимости от данных. Условные операторы позволяют выполнять разные блоки кода в зависимости от условия.

Без условий:
```python
# Программа всегда выводит одно и то же
print("Добро пожаловать!")
```

С условиями:
```python
age = 17

if age >= 18:
    print("Добро пожаловать!")
else:
    print("Вход только для совершеннолетних")
```

## Оператор if

Базовая форма — проверяет одно условие.

```python
if условие:
    # блок кода, выполняется если условие True
```

Важно: блок кода внутри `if` **обязательно** оформляется отступом (4 пробела или Tab).

```python
temperature = 35

if temperature > 30:
    print("Жарко!")
    print("Возьмите воду")

print("Хорошего дня!")  # Выполняется всегда
```

Вывод при `temperature = 35`:
```
Жарко!
Возьмите воду
Хорошего дня!
```

Вывод при `temperature = 20`:
```
Хорошего дня!
```

Условием может быть любое выражение, которое приводится к `bool`:

```python
name = "Иван"

if name:            # True, если строка не пустая
    print(f"Привет, {name}!")

balance = 0
if not balance:     # True, если баланс равен 0
    print("Пополните баланс")
```

## Оператор if-else

Добавляет ветку для случая, когда условие ложно.

```python
if условие:
    # выполняется если True
else:
    # выполняется если False
```

```python
age = 16

if age >= 18:
    print("Вы совершеннолетний")
else:
    print("Вы несовершеннолетний")

# Вывод: Вы несовершеннолетний
```

Ещё один пример:

```python
number = -5

if number >= 0:
    print(f"{number} — неотрицательное число")
else:
    print(f"{number} — отрицательное число")
```

Выполняется **строго одна** из двух веток — либо `if`, либо `else`.

## Оператор if-elif-else

Позволяет проверить несколько условий последовательно.

```python
if условие_1:
    # если условие_1 True
elif условие_2:
    # если условие_1 False, а условие_2 True
elif условие_3:
    # если условие_1 и 2 False, а условие_3 True
else:
    # если все условия False
```

Пример — оценка по баллам:

```python
score = 78

if score >= 90:
    grade = "Отлично"
elif score >= 75:
    grade = "Хорошо"
elif score >= 60:
    grade = "Удовлетворительно"
else:
    grade = "Неудовлетворительно"

print(f"Оценка: {grade}")  # Оценка: Хорошо
```

Важно: как только одно условие оказывается истинным, остальные `elif` и `else` не проверяются.

```python
x = 10

if x > 5:
    print("больше 5")   # Выполняется это
elif x > 3:
    print("больше 3")   # Это НЕ выполняется, хотя тоже верно
else:
    print("не более 5")
```

### Количество веток

- `elif` можно использовать сколько угодно раз
- `else` — необязателен и может быть только один, в самом конце

```python
month = 4

if month in [12, 1, 2]:
    season = "Зима"
elif month in [3, 4, 5]:
    season = "Весна"
elif month in [6, 7, 8]:
    season = "Лето"
elif month in [9, 10, 11]:
    season = "Осень"
else:
    season = "Неверный месяц"

print(f"Сейчас: {season}")  # Сейчас: Весна
```

## Тернарный оператор

Краткая форма `if-else` для простых случаев — записывается в одну строку.

```python
значение_если_True if условие else значение_если_False
```

```python
age = 20
status = "совершеннолетний" if age >= 18 else "несовершеннолетний"
print(status)  # совершеннолетний

number = -5
description = "положительное" if number > 0 else "неположительное"
print(description)  # неположительное
```

Тернарный оператор удобен, когда нужно присвоить одно из двух значений. Для сложной логики лучше использовать обычный `if-else`.

## Типичные ошибки

### Пропущенное двоеточие

```python
if age >= 18   # SyntaxError: пропущено двоеточие
    print("OK")
```

### Неправильный отступ

```python
if age >= 18:
print("OK")   # IndentationError: нет отступа
```

### Использование = вместо ==

```python
if age = 18:   # SyntaxError: это присваивание, а не сравнение
    print("OK")

if age == 18:  # Правильно
    print("OK")
```

### Лишний else после elif

```python
# Если все условия проверены явно, else может ввести в заблуждение
number = 0

if number > 0:
    print("Положительное")
elif number < 0:
    print("Отрицательное")
elif number == 0:
    print("Ноль")
# else здесь не нужен
```

## Практические примеры

### Калькулятор скидок

```python
total = 2500

if total >= 5000:
    discount = 0.20
elif total >= 2000:
    discount = 0.10
elif total >= 1000:
    discount = 0.05
else:
    discount = 0

final_price = total * (1 - discount)
print(f"Сумма: {total} руб.")
print(f"Скидка: {discount:.0%}")
print(f"К оплате: {final_price:.2f} руб.")
```

Вывод:
```
Сумма: 2500 руб.
Скидка: 10%
К оплате: 2250.00 руб.
```

### Определение времени суток

```python
hour = 14

if 6 <= hour < 12:
    greeting = "Доброе утро"
elif 12 <= hour < 18:
    greeting = "Добрый день"
elif 18 <= hour < 23:
    greeting = "Добрый вечер"
else:
    greeting = "Доброй ночи"

print(f"{greeting}!")  # Добрый день!
```
