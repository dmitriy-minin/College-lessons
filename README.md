# 🎓 EduPortal — Учебная платформа на GitHub Pages

Это минималистичный, быстрый и адаптивный портал для обучения студентов. Сайт построен на статическом генераторе **Jekyll** и оптимизирован для бесплатного хостинга на **GitHub Pages**.

## Стек 

Фронт: HTML, чистый CSS, SVG.

`Jakyll` - статический генератор сайтов (SSG), написанный на языке Ruby. Он превращает Markdown-файлы и HTML-шаблоны в готовый быстрый сайт;

`Liquid` - шаблонизатор от Shopify. Именно с его помощью реализована логику (циклы for, условия if, переменные site.python_lessons) внутри HTML.

`Markdown` - облегченный язык разметки. Позволяет писать контент уроков простым текстом, который автоматически конвертируется HTML.

## Особенности проекта

* **Модульность:** общие элементы (меню, футер) хранятся в папке `_includes`. Изменения в одном файле применяются ко всему сайту.
* **Автоматизация уроков:** для добавления нового урока достаточно создать `.md` файл в соответствующей папке коллекции.
* **Адаптивный дизайн:** сайт полностью оптимизирован под мобильные устройства (Media Queries + Meta Viewport).
* **Интерактивность:** плавные CSS-анимации появления блоков и эффекты наведения (Hover).
* **Скорость:** никаких тяжелых фреймворков. Чистый HTML и CSS.

## 📁 Структура файлов

```text
gitlessons/
├── _includes/             # Повторяющиеся части (header.html, footer.html)
├── _layouts/              # Шаблоны (default.html — общий, lesson.html — для уроков)
├── _python_lessons/       # Коллекция уроков по Python (Markdown файлы)
├── assets/                # Статика
│   └── style.css          # Основные стили, адаптивность и анимации
├── _config.yml            # Настройки Jekyll и регистрация коллекций
├── Gemfile                # Список зависимостей Ruby
├── index.html             # Главная страница со списком курсов
└── python.html            # Страница курса со списком всех уроков
```

## Быстрый старт

Для Linux
```
Установка зависимостей
```text
sudo apt update
sudo apt install ruby-full build-essential zlib1g-dev
```

Настройка пути
```
echo 'export GEM_HOME="$HOME/gems"' >> ~/.bashrc
echo 'export PATH="$HOME/gems/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

Установка Jekyll и зависимостей
```
gem install jekyll bundler
bundle install
```

Запуск сервера
```
bundle exec jekyll serve
```

## Как работать с контентом

**Добавление предмета:**

1. Регистрация в файле `_config.yml`(Важно: после этого нужно перезагрузить сервер);
2. Создание папки в корню проекта. Формат названия `_object_lessons`;
3. Добавление файлов с уроками:
   - Создать файл `lesson-01.md`,
   - Пример содержимого файла:
```
---
layout: lesson
title: "Введение в Object"
order: 1
---
Текст первого урока по ...
```
4. Создание страницы списка уроков в корню проекта, формат названия `object.html`(можно просто скопировать существующие файлы заменив название файла и коллекции);
5. В index.html найти нужную карточку, убрать у неё класс `locked` и заменить ссылку с `#` на `object.html`;
6. В `_includes/header.html` заменить ссылку в меню с `#` на `name.html` и убрать класс `disabled`.

**Добавить урок к существующему предмету**

1. Созать файл с названием `lesson-01.md`;
2. Поместить его в папку `_object_lessons`;
3. Проверить содержимое файла
```
---
layout: default
title: "Урок 1: Введение"
order: 1
---
# Добро пожаловать в Object
Здесь вы пишете текст урока, вставляете примеры кода и задания.
```
